# PS_Password_Generator
## Powershell Script to generate secure passwords
Execute o script e forneça o comprimento da senha que deseja.

Por padrão a senha é gerada com as seguintes classes de caracteres:
- Minúsculas a-z
- Maiúsculas A-Z
- Números 0-9
- Hífen -
- @?
- Underscore _
